# Repository Overview

This repository demonstrates the fundamentals of Java Database Connectivity (JDBC) by connecting a Java application to a MySQL database and executing SQL statements using the **Statement** interface.

The project is designed for:

- Students
- Beginners
- Java Developers
- College Projects
- Interview Preparation
- JDBC Fundamentals

# Learning Objectives

By studying this project, you will learn how to:

- Connect a Java application to a MySQL database.
- Configure the JDBC Driver.
- Establish database connections.
- Execute SQL statements using the Statement interface.
- Perform DDL operations.
- Perform DML operations.
- Manage database resources using Try-With-Resources.
- Handle SQL exceptions effectively.
- Understand the JDBC workflow from Java application to database.

# Recommended Learning Path

For the best learning experience, follow this order:

1. Read the main **README.md**.
2. Understand the JDBC architecture.
3. Explore the project structure.
4. Review the source code implementation.
5. Execute the project locally.
6. Experiment with different SQL statements.
7. Extend the project with CRUD operations.
8. Move on to PreparedStatement-based JDBC projects.
